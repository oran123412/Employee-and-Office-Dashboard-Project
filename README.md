# Employee and Office Dashboard

Flask + SQLite dashboard to manage employees and offices (with assignments), plus a REST API.

## Run

```bash
python app.py
```

- UI: `http://127.0.0.1:5000/`
- API: `http://127.0.0.1:5000/api/offices` / `http://127.0.0.1:5000/api/employees`

## Notes

- The SQLite database (`company.db`) is created/updated automatically when you run the app.
